# 权限管理系统
一个可以管理用户权限的前后端分离系统
# 开发工具
1. JDK 1.8
2. IDEA 2019.3.5
# 框架
1. Spring Boot 2.4.3
2. Mybatis-plus 3.2.0
3. Shiro
4. JWT
